## 国内安装版

这些版本往往简化了安装步骤，提供了更简单的入口。

[QClaw - 微信远程办公 AI 助手 | 腾讯出品](https://qclaw.qq.com/)
只有 Windows 与 Mac 版，目前已经全面开放，无需邀请码即可使用。

[WorkBuddy - AI Agent 办公新范式](https://copilot.tencent.com/work/)
只有 Windows 与 Mac 版，发现它与 CodeBuddy 都并入到了一个新的域名耶。

[360安全龙虾官网](https://claw.360.cn/)
目前只有 Windows 版，官网显示 Mac 版敬请期待。Windows 版虽然可以在 Win7 系统安装，但是安装后会无法运行哦，不建议尝试。

[LobsterAI - 有道 AI Agent 产品](https://lobsterai.youdao.com/#/index)
官网 logo 旁写的是有道龙虾，有 Windows 与 Mac 版。

[CoPaw — 协作个人智能体工作站 | 基于 AgentScope 的开源 AI 助手](https://copaw.bot/zh/)
阿里的，提供了 pip 命令到 Docker 和云端部署。

[JVS Claw](https://jvs.wuying.aliyun.com/)
阿里的，目前只提供 Mac 版的，但是也提供了云端部署。

[扣子编程，你的 AI 开发伙伴， Vibe Coding 基础设施，自然语言对话开发智能体、工作流、网页应用、移动应用，一键部署上线](https://code.coze.cn/home?openclaw=1)

## Agent 类

这个部分与衍生龙虾的区别，主要是它可以调用一些技能，部分产品还支持自定义技能，但是它们都不是衍生龙虾哦，无法直接使用龙虾技能，不能接入各类聊天软件。

[MiniMax Agent: 简单指令, 无限可能](https://agent.minimaxi.com/)
区别与云端龙虾 MaxClaw 产品，这个是完全免费的，每日登录即可获取 200 积分，各类任务根据实际情况消耗积分，但是要注意积分有效期，每日0点刷新。

[扣子，你的 AI 办公助手](https://www.coze.cn/)
目前主要分为扣子（原扣子空间）与扣子编程两个版块，扣子编程只有网页版的，原来的智能体也并入到这里了；扣子（空间）与手机App互通，每日登录即可获得1500积分，并且积分7天有效，扣子（空间）与扣子编程消耗的是同一个积分。
